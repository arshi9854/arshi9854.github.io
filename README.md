# Arshiya Shaik - Portfolio Website

A modern, interactive, and professional portfolio website showcasing my experience as a Software Engineer and AI Developer.

## Features

- **Responsive Design**: Fully responsive across all devices (desktop, tablet, mobile)
- **Smooth Animations**: Intersection Observer API for scroll-triggered animations
- **Interactive Navigation**: Smooth scrolling with active link highlighting
- **Modern UI**: Dark theme with gradient accents and glassmorphism effects
- **Performance Optimized**: Fast loading with minimal dependencies

## Sections

1. **Hero**: Eye-catching introduction with social links
2. **About**: Background, education, certifications, and awards
3. **Experience**: Professional timeline with achievements
4. **Projects**: Featured projects with technology stacks
5. **Skills**: Comprehensive technical skills organized by category
6. **Contact**: Multiple ways to get in touch

## Technologies Used

- HTML5
- CSS3 (Custom Properties, Grid, Flexbox, Animations)
- Vanilla JavaScript (ES6+)
- Font Awesome Icons

## Quick Start

1. Clone or download this repository
2. Open `index.html` in your browser
3. That's it! No build process required.

## Customization

To customize the portfolio:

1. **Colors**: Edit CSS variables in `styles.css` (`:root` section)
2. **Content**: Update text in `index.html`
3. **Animations**: Modify JavaScript in `script.js`
